Utility Functions for Graphics

Usage

mfrow(n)

Arguments

n
number of plots to be arranged.

Description

Utility Functions for Graphics

mfrow returns a 2-long numeric vector suitable to use in par(mfrow=x), that will arrange n panels in a single plot.

Examples

[1] 1 1

mfrow(2)



[1] 1 2

[1] 2 2
